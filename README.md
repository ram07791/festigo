# 🎉 Festigo – Event Management System

## 📌 Overview
**Festigo** is a full-stack web application designed to simplify event management and booking. It allows users to explore, book, and manage events seamlessly, while organizers can create and manage events efficiently.

---

## 🚀 Features

### 👤 User Features
- User registration and login system  
- Browse and search for events  
- Book events easily  
- View upcoming and past events  
- Secure session management  

### 🛠️ Admin / Organizer Features
- Create, edit, and delete events  
- Manage event schedules  
- View registered users and bookings  
- Dashboard for event management  

### 💡 Additional Features
- Responsive UI design  
- Image upload for events  
- SQLite/MySQL database support  
- Authentication and authorization  
- Flash messages for user feedback  

---
